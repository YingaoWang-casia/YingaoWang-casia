### 开源项目

#### Speech AI / Turn-taking

- TurnSense：面向真实人机语音交互场景的三分类轮次检测模型，将用户输入划分为 complete / incomplete / invalid 三类，用于判断系统应当立即响应、继续等待，还是忽略无效输入。
- CoDeTT Benchmark：面向全双工语音交互的多分类轮次检测 Benchmark，用于评测 Turn-Taking 模型在多场景决策任务中的表现；论文已投稿至 Interspeech。

#### LLM Agent / RAG / Codex Skill

- ProjectProof for Codex：面向 AI / Agent / RAG / 算法 / 数据 / 前后端实习项目的 Codex Skill，用于把真实项目重新放回证据链、工程边界和面试追问中，降低“小厂项目 / 内部项目 / Agent Demo”的 toy 感。
- InterviewProof-RAG：Codex-first 的本地面经 RAG 与项目压测系统，将零散面经整理为可追溯的 InterviewCard，并通过本地索引、Codex Pack 和 ProjectProof Bridge 反向检查个人项目能否扛住真实面试追问。
